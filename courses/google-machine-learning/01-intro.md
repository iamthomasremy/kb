# 01-intro

Course: https://developers.google.com/machine-learning/intro-to-ml/

## Concepts

- Supervised learning
- Unsupervised learning
- Reinforcement learning
- Generative AI

## Key ideas

Supervised learning:
- Regression model: predicts a numeric value
- Classification model: binary or multiclass

Unsupervised learning models: identify meaningful patterns in a dataset.

Reinforcement learning models: make predictions by getting rewards or penalties based on action performed.

Generative AI models: create content from user input.
